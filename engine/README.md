# Chess Engine Directory

This directory contains the Stockfish chess engine executables for different platforms.

## Expected Files

- **Linux**: `stockfish-linux-x64`
- **Windows**: `stockfish-windows-x64.exe`
- **macOS (Intel)**: `stockfish-macos-x64`
- **macOS (Apple Silicon)**: `stockfish-macos-arm64`

## How to Download

Run the setup script for your platform:

**Linux/macOS:**
```bash
./setup.sh
```

**Windows:**
```cmd
setup.bat
```

## Manual Download

1. Visit: https://stockfishchess.org/download/
2. Download the appropriate version for your platform
3. Extract and rename the executable according to the naming convention above
4. Place it in this directory

## Version

The setup scripts download **Stockfish 16.1** (latest stable release).
